# 总结
本次2050新生论坛返场活动聚焦AI领域，邀请5位嘉宾围绕AI安全、设计、创业、OPC实践和投资等方向进行分享，为参与者提供了多维度的见解和经验，内容如下：
- AI安全：从裸奔到受控
- Agent安全风险分类
- 过度授权风险：Adam指出，在Agent时代，过度授权是一个常见且易被忽视的安全问题。用户常将Agent定位为24小时在线助手，安装时习惯使用高权限，如root权限，且在拉群协作时缺乏安全基线设置，导致设备和对外通道权限过高，存在安全隐患。
- Prompt注入风险：这是AI时代新产生的安全风险。用户使用Agent处理信息时，若访问的网站包含肉眼不可见的恶意Prompt指令，Agent可能会执行这些指令，导致本地文件被盗或发送恶意邮件等问题。
- 恶意Skill风险：Skill生态开放，门槛低，匿名性强，黑客作恶成本低，导致恶意Skill猖獗。如Claude code上部分热门Skill存在隐藏恶意命令，窃取用户敏感信息和资产。
- 其他随机风险：还包括搜索引擎推荐恶意安装包、AI幻觉和系统漏洞等随机发生的攻击点，这些风险虽然不典型，但也需引起重视。
- 安全防范建议
- 最小授权原则：遵循安全方面的最小授权原则，避免给予Agent过高权限。
- 人工二次确认：对于敏感行为，如访问本地文件、编辑对外邮件、资金转账等，设置人工二次确认机制，确保操作经过用户确认。
- 使用安全工具：安装Agent Guard等安全工具，对Skill进行扫描，防范恶意Skill攻击，保障Agent使用过程中的安全。
- AI设计：品牌与IP的力量
- AI时代品牌的重要性
- 注意力稀缺：Lucky认为，随着AI产品技术成本下降，用户记住产品的成本上升，注意力成为AI时代唯一不会通胀的资产。
- 感性价值不可替代：用户选择产品更注重感觉，功能可被复制，但感觉无法复制。如豆包以其啰嗦但热情的形象和邻家女孩的IP，给用户留下独特的印象。
- AI产品品牌差异化破局
- IP是差异化的关键：AI产品品牌差异化难，功能易被追平，而IP是品牌差异性最快的挖掘机，能承载品牌理念，与用户建立情感连接。
- 案例分析：物起万象：以社区硬件板块的AI品牌物起万象为例，选择小象作为IP形象，与其他AI品牌形成差异化。小象的白色、缓慢和圆润的形象，体现了反叛和独特的理念，扩展到角色世界后，成为品牌与用户的情感API，实现用户主动传播。
- 品牌建设要点
- IP经营关系：品牌IP不是简单的图，而是经营品牌和用户之间关系的重要手段。
- 品牌与产品并行：品牌不是装饰，而是地基，应与产品相辅相成、并驾齐驱。
- 理性产品需要感性锚点：工具类和B端产品更需要有人格的温度，以满足用户的情感需求。
- AI创业：贴近普通人需求
- 普通人对AI的认知误区
- 无所不能又无关：大王指出，普通人认为AI无所不能，但又觉得与自己无关。短视频平台上AI的强大表现让普通人觉得AI像神技，但现实中AI应用多从工具出发，未解决实际问题，导致普通人难以参与。
- 创业应关注真实需求：创业核心是满足真实需求，能否有人购买、持续赚钱是关键。创业者应走出圈子，了解普通人的日常需求，避免在竞争激烈的领域内卷。
- 创业案例分析
- 知识付费转型：贵州一位原做知识付费卖资料的朋友，在AI时代将业务转向卖写文书的提示词，获得了市场认可，证明了贴近真实需求的创业方向是可行的。
- 智能菜品留样秤：校园食堂管理公司引入智能菜品留样秤，解决了食堂留样的痛点，不仅自己使用，还推荐给同行，体现了细分场景创业的价值。
- 小龙虾U盘：有人通过卖小龙虾U盘获得收益，说明创业机会无处不在，应从历史商业经验中寻找灵感。
- 创业技巧与建议
- 学习经典创业技巧：创业者可学习SWOT分析、商业画布等经典创业技巧，明确自身优势、劣势、机会和威胁，提高创业成功率。
- 明确目标与路径：目标要清晰，路径要明确，可将目标拆分为关键节点和具体任务，如打造AI自媒体账号可按定位、内容、增长、变现等节点进行拆解，降低焦虑，确保方向正确。
- 关注官方免费课程：当地团委、人社局会提供与创业相关的免费课程，创业者可关注学习。
- OPC实践：真实的创业之路
- OPC的概念与早期规划
- 一人公司的构想：志雄最初将OPC理解为一人公司，从功能角度出发，认为一个人能同时完成生产和销售，形成最小化的业务链路。
- 职场技能积累：他在北京花7年时间从程序员逐步晋升为项目经理和产品，掌握了软件生产端的各个环节，但发现还缺少销售环节。
- 创业过程的曲折与转型
- 销售工作的困境：回到武汉后，志雄进入一家营销软件公司做销售，但公司的旧模式和价值观与他产生冲突，最终裸辞。
- 创业方向的调整：尝试做企业信息化业务时，遇到技术执念和产品交付问题，意识到产品力不等于销售力，决定转向ToC业务，通过自媒体获客。
- 自媒体学习与实践：报名参加自媒体课程，虽线下课程内容与线上相似，但获得了心理上的激励，开始行动起来做自媒体账号，接触AI领域，搭建AI培训课。
- OPC的实现与感悟
- 商业机会的到来：参加社区AI训练营获得成就后，得到培训机构的合作邀请，将之前的课程落地，实现商业化。
- C端收益的获取：通过小红书发布教程和接单，解决学生认证和AI工具使用问题，获得收益。
- OPC的核心：OPC的关键是打通商业全流程闭环，通过自媒体构建个人品牌，建立客户信任，满足客户需求，实现交易。
- AI投资：把握产业机会
- AI产业发展趋势
- 资本关注焦点：Jack指出，AI产业正处于早中期，过去几年经历了多轮叙事发展，资本从最初对ChatGPT的争议，到如今真金白银地投入，带动了光模块等领域的造富神话。
- 产业发展阶段：目前上游AI算力基础设施建设已基本完成，下游AI应用方向需求逐渐显现，正处于产业发展的关键节点。
- 个人投资与创业机会
- 投资端机会：个人可通过找到合适的投资标的，参与AI产业发展，享受时代红利。
- 创业端机会：上游设备基建投资门槛高，个人难以参与，而应用软件和服务方向轻资产运营，个人可结合自身优势，在细分垂类领域创造新的AI应用。
- 垂直Agent方向：垂直Agent与细分领域数据和行业知识关联紧密，通用型Agent难以覆盖，是个人参与的机会点。如阿法派公司结合AI和金融细分领域，打造了围绕投资研究人员的AI软件产品。
- 创业模式与建议
- 成功路径：很多人通过制造话题吸引用户，提供服务或数据库，逐步向产品方向转型。国外多选择APP，国内可从小程序开始，逐步发展到网页端和APP端。
- 格局放大：在AI时代，应格局放大，将自身经验或他人经验包装成工作流Skill，打造AI产品，实现更大价值。
- 后续工作计划
- 资料分享：将各位嘉宾的PPT以PDF格式分享到微信群，方便参会者进一步学习和交流。

- 社群交流：欢迎大家加入新生论坛返场群，在群里继续交流AI相关问题，分享经验和见解。
